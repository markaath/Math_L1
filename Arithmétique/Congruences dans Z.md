
**Définition** :
Soient $(a,b,c)\in\mathbb Z³$
Alors on dit que $a$ est congru à $b$ modulo $c$ si $b$ est le reste de la division euclidienne de $a$ par c et on écrit $a\equiv b \pmod c$ 

**Propriété** :
Soient $(a,b,c)\in\mathbb Z³$
$a\equiv b \pmod c \Leftrightarrow c|(a-b)$ 
*Preuve* :
$a\equiv b \pmod c\Longleftrightarrow \exists k\in\mathbb Z : a = kc +b$
$\Longleftrightarrow \exists k\in\mathbb Z : a-b = kc$
$\Longleftrightarrow c|(a-b)$

**Propriétés** :
Soient $(a,b,c,d,n)\in\mathbb Z⁵$
1) $a\equiv b \pmod n \Rightarrow b\equiv a \pmod n$
2) $a\equiv b\pmod n \;\land\; b\equiv c\pmod n \Rightarrow a\equiv c\pmod n$
3) $a\equiv b \pmod n \;\land\; c\equiv d\pmod n \Rightarrow (a+c)\equiv (b+d) \pmod n$
4) $a\equiv b \pmod n \;\land\; c\equiv d\pmod n \Rightarrow ac\equiv bd \pmod n$
5) $a\equiv b\pmod n \;\land\; c\in\mathbb Z \Rightarrow ac \equiv bc \pmod {cn}$
6) $a\equiv b\pmod{cn} \Rightarrow a\equiv b \pmod n$
7) $a\equiv 0\pmod c \;\land\; b\equiv 0 \pmod d \Rightarrow ab\equiv 0 \pmod {cd}$
